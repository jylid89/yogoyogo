# yogoyogo

## 한국소프트웨어기술진흥협회 JAVA Framework & 안드로이드 개발자 양성과정 프로젝트

- 김상준
- 김예은
- 김현진
- 이종엽

## 프로그램 개요
#### 푸드트럭 사업자, 고객 간 소통가능한 커뮤니티
#### 사업자에게 홍보의 도구가 되고 고객은 다양한 푸드트럭의 정보를 알 수 있다
